# Birko.Data.MongoDB.ViewModel

MongoDB ViewModel repository implementations for the Birko Framework.

## Features

- ViewModel repositories for MongoDB
- Index creation support
- MongoDB-specific query methods

## Installation

```bash
dotnet add package Birko.Data.MongoDB.ViewModel
```

## Dependencies

- Birko.Data
- Birko.Data.ViewModel
- Birko.Data.MongoDB
